Ejercicio 1

Se trata de escribir un script que diga si un número es par o es impar.Recordemos que un número es par si al dividirlo entre 2 da como resultado 0.

  • Primero necesitamos indicarle al usuario que digite un numero entero
  • Usamos let porque cambia de valor
  • parseInt para cambiar el valor a entero
  • prompt para pedir al usuario que digite un numero
  • let para guardar lo que escribio el usuario
  • % (MOD) O RESTO DE DIVISIÓN para que nos de 0 y como consecuencia saber si es par o impar

Dentro de la estructura if

  • En los parentesis se escribe la condicion-accion que se llevara a cabo
  • En este caso queremos se escribe la variable par sea igual a 0
  • Si es residuo es 0 pasa a la siguente instruccion del alert
  • Osease el num(lo que escribio el usuario) y se concatena "+" con es par
  • sino pasa al siguiente alert de es impar

let num = parseInt(prompt("Teclea un numero entero "));

let par = num % 2;

   if (par === 0) {

       alert(num +" es par")

   }else{

       alert(num +" es impar")

   }
